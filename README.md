# Central Hub — Task Manager & Nessus Scan Comparison

An interactive static web toolkit containing two utility applications:
1. **Task Manager:** A clean task organization board to create, filter, complete, and track prioritize tasks.
2. **Nessus Comparison Tool:** A utility to upload multiple Nessus XML vulnerability scan files, compare them, and highlight differences in severity levels and patch updates.

## Live Demo
Check out the live deployment here:
👉 **[https://kundanchauhan.github.io/taskManger/](https://kundanchauhan.github.io/taskManger/)**

## Features
- **Modern Glassmorphism UI:** Built with dark theme aesthetics matching modern portfolio styles.
- **Task Management Board:** Add tasks, set priority labels (High, Medium, Low), filter by completion, and search instantly.
- **Nessus Vulnerability Auditing:** Upload Nessus scan results to parse and generate comparison metrics.

## Tech Stack
- Frontend: HTML5, CSS3 (Custom properties, grid, animations)
- Scripting: Vanilla ES6+ JavaScript (LocalStorage for task persistence, XML DOM Parser for Nessus reports)
